Main Content

Moderation

    s!addrole (ROLE NAME) (USER)- This adds a role to a user.

    s!ban (USER) - This command bans the user mentioned

    s!dm (USER) (MESSAGE) - This dm’s the user mentioned whatever is put in the message.

    s!id (USER) - Gives the mentioned user’s ID.

    s!kick (USER) - Kicks the mentioned user.

    s!mute (USER) - Mutes the mentioned user.

    s!purge (AMOUNT)- Purges a certain amount of messages under 100 and under 14 days of age.

    s!removerole (ROLE) (USER) - Removes a role from the mentioned user.

    s!report (USER) - A useful report system, all reports go to the #silent-log channel.

    s!say (MESSAGE) - Says whatever you want it to.

    s!support (MESSAGE) - Send a support ticket to the support team, if abused you will be blacklisted and the bot will be removed from your server.

    s!unban (USER) - Unbans banned user.

    s!warn (USER) (MESSAGE) - Warns a user.

Info

    s!avatar (USER) - Posts the avatar of the mentioned user.

    s!invite - Creates an invite.

    s!emojis - Displays all the servers emojis (WARNING: May lag some users if there is a lot and a lot of animated emojis!!)

    s!help - Gives all commands and the bot’s patreon link.

    s!info - Displays information about the bot.

    s!invitestats - Displays all invites in the server and the number of users invited with it. (Working on it, kinda messy!)

s!roles: See servers role list s!serverinfo: View server information like member count, bot count, discord partner etc. s!servers: View all the servers the bot is in s!stats: View bot stats like total users, servers, etc. s!uptime: View how long the bot has been online s!urban: Look something up on urban dictionary s!whois: View userinfo like their ID, username, icon, etc.
